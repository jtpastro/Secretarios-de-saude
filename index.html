Duração de mandato nas Secretarias Estaduais de Saúde

uma análise de como o gênero afeta o tempo dos mandatos nas regiões brasileiras

Alexandra Emanuela Novais Silva

2024-09-04

Apresentação

Nesse documento, nós analisamos a duração de mandato de secretários de saúde estaduais do Brasil durante 10 anos, de 2006 a 2016. Esse cargo faz parte da burocracia de alto escalão subnacional e é comissionado, ou seja, indicado pelos governadores estaduais. Cada estado possui um secretario estadual desaúde. É importante ressaltar que,a base de dados foi construída manualmente pelos autores, por meio de levantamento documental. Isso ocorreu devido a ausência de uma base consolidada disponível para acesso público.

Selecionando dados de intesse

Aqui definimos as variáveis de duração dos mandatos como númericas, criamos a coluna duração do mandato e criamos a coluna Regiao. A última servirá para agregarmos dados de todas as regiões.

prepare_data <- function(df, nome_regiao) {df %>% 
mutate( Mandato_Inicio = 
as.numeric(Mandato_Inicio), Mandato_Final = as.numeric(Mandato_Final), 
duracao_mandato = (Mandato_Final - Mandato_Inicio) + 1 ) %>% 
select(Gênero, duracao_mandato) %>% mutate(nome_regiao)}

Também juntamos os dados das regiões para posteriormente apresentarmos os dados conjuntos por região de forma agregada.

secretarios_combinados <- bind_rows(
  prepare_data(Secretarios_de_saude_CO, "Centro-Oeste"),
  prepare_data(Secretarios_de_saude_Norte, "Norte"),
  prepare_data(Secretarios_de_saude_Nordeste, "Nordeste"),
  prepare_data(Secretarios_de_saude_Sudeste, "Sudeste"),
  prepare_data(Secretarios_de_saude_Sul,"Sul"))

Agora omitimos dados que que possuem valores ausentes:

secretarios_combinados <- na.omit(secretarios_combinados)

Intervalo de Confiança

Agora criamos um dataframe chamado intervalo de confiança com as variáveis: media da duração do mandato; variância da duração do mandato; a raiz da variância, que é igual ao desvio padrão; o n, que refe-se ao tamanho da amostra;e o erro padrão.Todas essas são calculadas referentes a cada região e ao gênero.

intervalo_confiança <- secretarios_combinados %>%
  group_by(Gênero, nome_regiao) %>%
  summarise( media = mean(duracao_mandato, na.rm = TRUE),       
    variancia = var(duracao_mandato, na.rm = TRUE),   
    raiz_variancia = sqrt(variancia),   
    n = n(),                             
    erro_padrao = sqrt(variancia) / sqrt(n))

Calculando a Margem de Erro

Aqui criamos uma coluna para margem de erro dentro do dataframe e o calculamos.

margem_erro <- intervalo_confiança %>%
  mutate(
    margem_erro = qt (0.975, df = n - 1) * erro_padrao)

Calculando os limites inferior e superior e criando uma coluna para margem de erro.

intervalo_confiança <- margem_erro %>%
  mutate(
    limite_inferior = media - margem_erro,  
    limite_superior = media + margem_erro)
